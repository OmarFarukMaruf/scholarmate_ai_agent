# scholarmate_ai_agent
The AI Scholarship Finder Agent is an autonomous agent that collects a student’s academic background, performs real-time web search, identifies globally available scholarships, and stores them in a structured format (CSV) for later use.

This project is built using the Google Agent Development Kit (ADK) and follows the best practices learned from the 5-Day Google + Kaggle AI Agent Course.

# Features
1. The agent collects student attributes such as:
	•	Last completed education (BSc/MSc/PhD)
	•	IELTS/TOEFL scores
	•	Publications
	•	Work experience
	•	Country and region
	•	Academic achievements
2. It then performs automated Google Search queries to identify relevant scholarships worldwide.
3. Filter & extract scholarship details
4. All matching scholarships are stored inside a CSV file for later use, filtering, manual reviewing
